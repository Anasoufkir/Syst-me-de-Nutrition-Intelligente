# Système de Nutrition Intelligente

Application capable de générer des recommandations nutritionnelles personnalisées,
scientifiquement fondées et réalistes, à partir des données utilisateur.

## Objectifs (V1)
- Validation du profil (UC-01)
- Calculs : IMC (UC-02), BMR (UC-03), TDEE (UC-04)
- Ajustement calorique sécurisé (UC-05)
- Macros : protéines (UC-06), lipides (UC-07), glucides (UC-08)
- Sortie structurée et explicable (non médical)

## Disclaimer
Ce système est éducatif et préventif. Il ne remplace pas un avis médical.
