# UC-01 — Validation du profil utilisateur

Objectif : valider les données (cohérence, bornes, champs obligatoires)
afin de permettre une analyse fiable.

Champs obligatoires :
- sexe, âge, taille, poids, niveau d’activité, objectif

Validations :
- valeurs numériques cohérentes
- champs non vides
- cohérence taille/poids/âge
